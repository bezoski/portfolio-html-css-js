# My First Portfolio 🚀

A short, responsive portfolio website created to learn and practice basic front-end technologies. This is my first project built completely from scratch using HTML, CSS, and JavaScript. 

The project features a modern dark theme and a custom bottom navigation bar.

## 🔗 Live Demo
You can check out the live version here: **[Live Demo](https://bezoski.github.io/portfolio-html-css-js/main.html)**

## 🛠️ Technologies
* **HTML5**
* **CSS3**
* **JavaScript**
